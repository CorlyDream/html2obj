努力进化，为的就是在这个市场里站着把钱赚了

发表于
收录于合集

开盘前降息，1年期LPR下调5个基点，5年期以上LPR下调15个基点，从4.45%降为4.3%。略有超预期的意味，因为5年期以上降得比1年期多...


5年期以上贷款主要去向是基建、房贷、大型项目建设等。这里面和所有人相关的当然是房贷，假设贷款100万的话，未来每年可以少还1500元。对于已经扛着房贷的，相当于每年多了1500元钱消费；对于打算买房的，算是多少给了点勇气。


长期看LPR还有下降空间，未来我国的无风险收益水平大概也就在3%左右。所以现在银行年化3%以上的存款都限额，一出来就被无脑抢空。你们不要以为只有大妈才会抢大额存单，前几天和银行的哥们聊天，学到了很多奇怪的知识。


比方说前两年年轻人里面流行炒各种莫名其妙的东西，大部分人都被套进去了，但也有少数人逃顶。拿着钱也不知道该干什么，最后都买了3%的银行存单。


(⊙o⊙)…，这说明大部分人的风险偏好很奇怪，总是在贪婪和恐惧两头摇摆。莫名其妙大赚一笔后就会变得保守，但也有可能保守了一辈子突然想要去冒险。


从自身修养来说，炒股这件事的本质就是保持一个合理的风险偏好，不要随着市场摇摆。


老头说“别人恐惧的时候我贪婪，别人贪婪的时候我恐惧。”


其实老头自身的风险偏好是基本不变的，只不过整个市场的情绪（风险偏好）摇摆幅度太大。以市场为参照物看老头，显得与众不同而已...


不管怎么说，降息这件事利空吃息差的金融行业，利好其他所有行业。今天全市场成交额1.03万亿，中位数涨幅0.41%，算是稳稳的幸福。


今天比较有意思的则是：


上证50      +0.21%

沪深300    +0.76%

中证500    +1.27%

中证1000   +0.75%

国证2000  +0.68%


上证50里金融股比较多属于被拖累了，其他指数涨幅差不太多。说明至少在今天，大盘股、中盘股、小盘股的走势比较均衡，小盘股骑在大盘股身上作威作福的PUA惨剧没有上演。大蓝筹爱好者算是迎来了一天的喘息机会...


......


1、虽然有利好，地产股今天的表现比较一般，并没有明显强于大盘。地产股的主要玩家是机构，正在赛道里玩得嗨，可能要看到实质性数据改善才会下场。但债市已经反应过来了，大部分地产债今天涨幅都不错...


2、海南表示疫情已经出现明显拐点，主要传播链基本阻断。我算了下日子，大概前后半个月左右时间。看了下，中免果然反弹...


3、士兰微上半年业绩5.99亿，同比+39.12%。看着还可以，但二季度环比一季度是下降的，半导体行业进入下行周期的趋势比较明显，前期爆炒的Chiplet概念今天也血亏。


4、克明食品半年业绩+89%，扣非+258%。这是因为股民上半年吃面特别多么？上面是开玩笑的，真实原因是去年的业绩非常差，所以今年显得特别好。


......


场子里就这些事，但场子外发现了个好玩的。


今年7月12日，有人举报上海中考数学题泄露，经过警方调查情况如下。


A和B的女儿要中考，请求在印刷厂工作的好哥们C偷窃试题，C就偷带手机拍摄了数学题传给A。


A手抄后将题目给女儿做，抄错了2题，导致女儿怎么都做不出来。


B的父亲D是退休数学老师，B就直接把题目给D做，但未告知是真题。


D当然也做不出来，就发给E帮忙。


E也做不出来，就找了高手F帮忙。


F也是数学老师，随手就把题目发到学生群要求学生完成。但后来发现题目是错的，就直接撤回。


群里有学生参加中考后发现很像真题，遂把截图传到了各种QQ群里，最终引发了举报。


呃，这明明是一件很严肃的事，但我看了只想笑。脑海里突然冒出了很多电影，都是那种小人物莫名其妙就干了一票大的那种。


因为是上海发生的事，角色我都给脑补好了，就徐峥领衔主演吧，《爱情故事》原班人马可上。有人说让雷佳音演那个把题目发到群里的数学老师，我认为不妥。因为雷佳音是东北人，演不出上海人的气质。
